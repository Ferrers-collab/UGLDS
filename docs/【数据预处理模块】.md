## 二、数据预处理方案

### 2\.1 缺失值处理

**检测结果**：
通过遍历订单信息表（2169 行），发现以下记录的重量或体积字段为空值：
订单编号 61、75、143、194、223、260、302、357、464、534、618、683、741、883、1038、1149、1226、1337、1474、1539、1625、1677、1721、1815、1890、1955、2021、2093、2166。

**处理方案**：

- **重量缺失**：采用**同客户均值填充法**。计算该客户其他订单的平均重量，作为缺失值的估计。若该客户仅此一单缺失，则取全局均值。

- **体积缺失**：同理采用**同客户均值填充法**。

**论文表述**：“对于附件订单信息中部分订单的体积或重量缺失，本文基于同客户历史订单的均值进行填补，考虑了客户需求的内部一致性，避免了全局均值填充可能引入的偏差。”

### 2\.2 数据转换与特征工程

#### 2\.2\.1 订单数据聚合

将 2169 条订单按**目标客户编号**汇总，得到每个客户的**总需求重量**和**总需求体积**。

#### 2\.2\.2 绿色区标记

计算各客户点到市中心 \(0,0\) 的欧氏距离：
$d_i = \sqrt{(X_i - 0)^2 + (Y_i - 0)^2}$
若 $d_i \leq 10$ km，则将该客户标记为**绿色区内客户**（`is\_green=1`），否则为绿色区外客户（`is\_green=0`）。
**验证**：题中明确给出 “30 个客户位于绿色配送区内”，预处理后需核验标记数量是否为 30。

#### 2\.2\.3 时间窗数值化

将时间窗从`hh:mm`格式转换为**以分钟为单位的连续数值**（自 0:00 起算），便于模型计算到达时间与惩罚成本。

#### 2\.2\.4 速度时段映射函数

建立速度分布查找表：

|时间段|开始 \(min\)|结束 \(min\)|μ\(km/h\)|σ\(km/h\)|
|---|---|---|---|---|
|拥堵 1|480\(8:00\)|540\(9:00\)|9\.8|4\.72|
|顺畅 1|540\(9:00\)|600\(10:00\)|55\.3|0\.12|
|一般 1|600\(10:00\)|690\(11:30\)|35\.4|5\.22|
|拥堵 2|690\(11:30\)|780\(13:00\)|9\.8|4\.72|
|顺畅 2|780\(13:00\)|900\(15:00\)|55\.3|0\.12|
|一般 2|900\(15:00\)|1020\(17:00\)|35\.4|5\.22|

#### 2\.2\.5 载重修正因子计算

能耗修正公式（线性插值）：
$FPK_{actual} = FPK_{base} \times \left(1 + 0.4 \times \frac{load}{capacity}\right)$
$EPK_{actual} = EPK_{base} \times \left(1 + 0.35 \times \frac{load}{capacity}\right)$
处理为：空载修正因子 = 1\.0，满载修正因子 = 1\.4（燃油）/1\.35（电动）。

### 2\.3 异常值检测

**方法**：

- **Z\-score 法**：对客户需求重量和体积计算 Z\-score，标记$|Z| > 3$的客户。

- **规则检验**：距离矩阵值非负、对角线为 0；时间窗满足开始时间 \&lt; 结束时间；速度分布参数合理。

**检测结果**：

- 订单重量和体积均无统计异常值。

- 距离矩阵严格对称，对角线为 0，所有值≥0。

- 所有时间窗均满足开始 \&lt; 结束。

- 数据质量整体较高。

### 2\.4 数据标准化 / 归一化

**若涉及多目标优化**（问题 2），成本和碳排放量纲差异大，可采用**Min\-Max 归一化**：
$x' = \frac{x - x_{min}}{x_{max} - x_{min}}$

**论文表述**：“在 NSGA\-II 多目标优化中，为避免目标量纲差异影响拥挤度计算的公平性，本文对适应度值进行 Min\-Max 归一化处理后，再计算拥挤度距离。”

### 4\.3 Python 代码（data\\\[\_preprocessing\.py\]\(\_preprocessing\.py\)）

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

DATA_DIR = "数据/"
OUTPUT_DIR = "预处理结果/"

df_orders = pd.read_excel(DATA_DIR + "订单信息.xlsx")
df_distance = pd.read_excel(DATA_DIR + "距离矩阵.xlsx")
df_coords = pd.read_excel(DATA_DIR + "客户坐标信息.xlsx")
df_timewin = pd.read_excel(DATA_DIR + "时间窗.xlsx")

df_orders['重量'] = df_orders.groupby('目标客户编号')['重量'].transform(lambda x: x.fillna(x.mean()))
df_orders['体积'] = df_orders.groupby('目标客户编号')['体积'].transform(lambda x: x.fillna(x.mean()))
df_orders['重量'].fillna(df_orders['重量'].mean(), inplace=True)
df_orders['体积'].fillna(df_orders['体积'].mean(), inplace=True)

customer_demand = df_orders.groupby('目标客户编号').agg(总重量_kg=('重量','sum'),总体积_m3=('体积','sum')).reset_index()
customer_demand.rename(columns={'目标客户编号':'客户编号'},inplace=True)

df_coords['距市中心_km'] = np.sqrt(df_coords['X (km)']**2 + df_coords['Y (km)']**2)
df_coords['绿色区'] = (df_coords['距市中心_km']<=10).astype(int)

def time_to_minutes(t_str):
    h,m=map(int,str(t_str).split(':'))
    return h*60+m
df_timewin['最早_min']=df_timewin['开始时间'].apply(time_to_minutes)
df_timewin['最晚_min']=df_timewin['结束时间'].apply(time_to_minutes)

df_customers=df_coords[df_coords['类型']=='客户'][['ID','X (km)','Y (km)','绿色区']].copy()
df_customers.rename(columns={'ID':'客户编号','X (km)':'X_km','Y (km)':'Y_km'},inplace=True)
df_attributes=df_customers.merge(customer_demand,on='客户编号',how='left')
df_attributes=df_attributes.merge(df_timewin[['客户编号','最早_min','最晚_min']],on='客户编号',how='left')

df_attributes.to_csv(OUTPUT_DIR+'customer_attributes.csv',index=False,encoding='utf-8-sig')
customer_demand.to_csv(OUTPUT_DIR+'customer_demand_clean.csv',index=False,encoding='utf-8-sig')
df_distance.to_csv(OUTPUT_DIR+'distance_matrix.csv',index=False,encoding='utf-8-sig')

plt.rcParams['font.sans-serif']=['SimHei']
plt.rcParams['axes.unicode_minus']=False
fig,ax=plt.subplots(figsize=(12,10))
circle=plt.Circle((0,0),10,color='green',fill=False,linestyle='--')
ax.add_patch(circle)
green=df_attributes[df_attributes['绿色区']==1]
other=df_attributes[df_attributes['绿色区']==0]
ax.scatter(green['X_km'],green['Y_km'],c='green',s=50,label='区内客户')
ax.scatter(other['X_km'],other['Y_km'],c='orange',marker='^',s=50,label='区外客户')
ax.scatter(20,20,c='red',marker='*',s=300,label='配送中心')
ax.set_title('客户空间分布及绿色配送区')
ax.legend()
plt.savefig(OUTPUT_DIR+'客户空间分布.png',dpi=300)
```
